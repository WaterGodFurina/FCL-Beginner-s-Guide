# Minecraft 光影（Shaders）

光影（Shaders）是 Minecraft 里的一种视觉效果增强包。装上之后，游戏里的阳光、水面、阴影、天空都会变得更真实，甚至能媲美一些 3A 大作的画面。

[detail]

## 光影到底改了什么？

原版 Minecraft 的画面比较"朴素"：方块棱角分明，水面是半透明的蓝色平面，太阳就是一个发光的圆饼。

装上光影之后，你会看到：

- 真实的水面：水会反光、会波动，能看到水下的折射效果。
- 动态阴影：人物、树木、建筑会投下真实的影子，随着太阳位置变化。
- 体积光/丁达尔效应：阳光穿过树叶、窗户时，能看到一道道光柱。
- 更好的天空：日出日落有霞光，晚上有银河星空。
- 全局光照：方块表面会有更柔和、更真实的明暗过渡。

简单说，光影就是给 Minecraft 换了一套"渲染引擎"，让画面从"卡通风格"变成"写实风格"。

## 光影对性能的影响

光影不是免费的午餐，它需要你的显卡（GPU）进行大量额外计算。

### 电脑上的情况

- 低端电脑/核显：装光影后帧数可能从 60 帧暴跌到 10-20 帧，甚至幻灯片。
- 中端显卡（如 GTX 1650 / RTX 3050）：可以跑中低配光影，帧数大概 40-130 帧。
- 高端显卡（如 RTX 4090）：可以流畅跑高配光影，甚至开启光追（RTX）。

:::tip 小贴士
光影主要吃显卡，对 CPU 和运行内存要求相对没那么高。
:::

## 手机 Java 启动器能跑光影吗？

能跑，但非常吃手机性能，不同机型差距极大。

手机 Java 启动器（如 Fold Craft Launcher 等）本质上是在手机上模拟运行电脑版的 Minecraft Java 版。这个过程中，手机要额外承担很多本不该它干的活

### 为什么手机上跑光影很吃力？

| 问题 | 原因 |
|------|------|
| 没有独立显卡 | 手机的 GPU 集成在芯片里，性能远不如电脑独显，跑光影时压力极大。 |
| 图形接口转换 | 电脑版 Minecraft 用 OpenGL 渲染，而手机只支持 OpenGL ES。启动器需要实时把 OpenGL 的指令"翻译"成 OpenGL ES，这个过程本身就要吃掉不少性能。 |
| 散热瓶颈 | 光影会让 GPU 满载，手机很快发烫，然后强制降频，帧数断崖式下跌。 |
| 兼容性问题 | 部分光影包经过 OpenGL ES 转换后，会出现水面变黑、天空花屏、阴影闪烁等渲染错误。 |

### 实际体验参考

| 手机档次 | 代表芯片 | 光影表现 |
|---------|---------|---------|
| 旗舰（骁龙 8 Elite 2 / 天玑 9500 等） | 最新一代旗舰芯 | 可以流畅跑中低配光影（如 Sildur's Vibrant Lite、BSL Lite），帧数能到 50-100 帧，但发热和耗电依然明显。 |
| 次旗舰（骁龙 8 Gen 2/3、天玑 9200+ 等） | 近两代旗舰芯 | 能跑，但不算流畅。选较低配的光影，帧数大概 25-60 帧，发热较大，建议搭配散热背夹。 |
| 中端（骁龙 7 系、天玑 8000 系等） | 中端芯片 | 能进游戏，但体验很差。帧数 15-30 帧，发热严重，玩起来像幻灯片。 |
| 低端/老机型 | 入门芯片 | 只有一两帧，不建议尝试。 |

:::tip 给手机玩家的建议
如果你用的是近两年内的旗舰或次旗舰手机，可以大胆尝试光影，但建议：
- 选 Lite / Low 版本的光影（中低配光影）。
- 配个散热背夹，避免过热降频。
- 插电玩，因为耗电会很快。

如果你的手机是中端或更低，建议彻底放弃光影，先把游戏本身跑流畅再说。
:::

## 选择建议

| 你的设备 | 建议 |
|---------|------|
| 电脑，有独显 | 可以大胆尝试一些中高配光影 |
| 电脑，核显/轻薄本 | 建议仅尝试低配光影，优先保证游戏流畅。 |
| 手机，旗舰/次旗舰 | 可以尝试低配光影，但要做好发热和耗电的心理准备。 |
| 手机，中端及以下 | 不建议装光影。能把游戏本身跑流畅就已经很不错了。 |

## 光影和 OptiFine / Iris 的关系

光影不能单独安装，需要配合光影加载器：

- OptiFine：老牌工具，自带光影支持。安装简单，但和其他模组兼容性一般。
- Iris + Sodium：新兴组合，Iris 负责加载光影，Sodium 负责优化性能。比 OptiFine 帧数更高，推荐新玩家使用。

小白记住：装光影之前，先确保你的游戏已经装了 OptiFine 或 Iris，否则光影文件放进去也没用。

:::note 注意
光影是"锦上添花"，不是"雪中送炭"。如果你的游戏本身就不流畅，装光影只会让情况更糟。先保证能流畅玩，再考虑好不好看。
:::
