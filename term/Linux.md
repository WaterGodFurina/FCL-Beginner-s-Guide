# Linux

Linux 是一种操作系统，和 Windows、macOS 并列，都是用来管理电脑硬件、让你能使用软件的"大管家"。

[detail]

不过，Linux 和 Windows 的"脾气"很不一样。

## Linux 和 Windows 有什么区别？

1. Windows 是"一家独大"，Linux 是"百花齐放"

Windows 只有微软一家在做，你买的电脑预装的基本都是它。但 Linux 不一样——它的核心代码是公开的，任何人都可以拿来改造，做出自己的版本。所以 Linux 有很多"版本"，每个版本都有自己的名字和特点。

2. Windows 要花钱（或预装在电脑里），Linux 基本免费

大多数 Linux 版本都可以免费下载、免费使用，不用买激活码。

3. Windows 界面统一，Linux 各有各的样

Windows 开机就是桌面、开始菜单、任务栏，大家都一样。但 Linux 不同版本的外观和操作习惯可能差别很大，有的像 Windows，有的像 macOS，有的干脆就是纯文字界面（没有鼠标点来点去，全靠打字）。

## 常见的 Linux 版本

虽然 Linux 版本很多，但你可能听到的主要是这几个：

| 名字 | 介绍 |
|------|-----------|
| **Ubuntu** | 最出名的"新手友好版"，界面像 Windows，社区活跃，教程多，很多人第一次接触 Linux 就是用它。 |
| **Debian** | Ubuntu 的"老祖宗"，更稳定、更保守，服务器上用得很多。 |
| **CentOS** | 以前很多企业服务器用它，现在地位有点尴尬（被红帽公司调整了策略）。 |
| **Fedora** | 红帽公司的"试验田"，新功能先在这里测试，适合想尝鲜的人。 |
| **Arch Linux** | 极简主义，安装和配置都要自己动手，被称为"极客版"，小白慎入。 |
| **Kali Linux** | 专门用来做网络安全测试的，普通人用不到。 |


:::tip 小贴士
对小白来说，如果真想试试 Linux，Ubuntu 是最稳妥的选择，网上教程最多，遇到问题最容易搜到答案。
:::

## Linux 藏在哪？

你可能从来没主动装过 Linux，但它其实无处不在：

- 安卓手机的底层：你用的 Android 系统，底层核心就是 Linux。可以说，Linux 是 Android 的"地基"。
- 网站和 App 的服务器：你刷的短视频、逛的淘宝、玩的网游，背后跑的服务器很多都用 Linux，因为它稳定、省资源。
- 路由器、电视盒子、智能家居：很多嵌入式设备里面跑的都是精简版的 Linux。
- 程序员的电脑：很多做开发的人喜欢用 Linux，因为工具链齐全、自由度高。

## 小白接触的场景

如果你只是普通用户，你大概率不会直接用到 Linux 桌面版。你可能接触 Linux 的场景只有这些：

- 买了台二手服务器/VPS，需要远程管理。
- 想折腾路由器、树莓派之类的设备。
- 学习编程，发现很多教程让你装 Linux。
- 看到某个软件只有 Linux 版本，Windows 跑不了。

:::note 注意

Linux 有很多版本，但版本之间并不完全兼容。能在 Ubuntu 上装的软件，不一定能直接装在 CentOS 上；某个教程里的命令，换个版本可能就不一样。所以看教程时，先确认对方用的是哪个版本。

:::

它和 Windows 最大的不同：Windows 一家独大、界面统一；Linux 版本众多、有的免费、有的面向服务器、有的面向新手。
