# Minecraft 模组（Mod）

模组（Mod，全称 Modification）就是玩家或开发者给 Minecraft 做的"外挂补丁"。装上之后，游戏会多出原版没有的功能、物品、生物，甚至完全变成另一个游戏。

[detail]

## 模组到底能干什么？

模组的功能五花八门，大致可以分成这几类：

1. 优化类——让游戏更流畅

原版 Minecraft 的代码写得比较"保守"，模组可以帮它"松松绑"：

- Sodium：大幅提升帧数，中低配电脑的救星。
- Lithium：优化游戏内部的运算逻辑，减少卡顿。
- Iris：让游戏支持光影（配合 Sodium 用，比 OptiFine 更流畅）。
- FerriteCore / ModernFix：减少内存占用，装了很多模组也不容易闪退。

2. 功能类——让游玩更方便

- 小地图（Xaero's Minimap / JourneyMap）：右上角显示地图，还能标记坐标。
- JEI（Just Enough Items）：按 R 键查看物品合成表，再也不用手动查 Wiki。
- 一键整理（Inventory Profiles Next）：背包乱糟糟？一键自动排序。
- 死亡坐标记录：死了之后自动告诉你尸体在哪，不用满世界找。

3. 内容类——让游戏更丰富

这是模组最吸引人的地方，相当于给 Minecraft 做 DLC：
- 工业类：加入机器、电线、自动化流水线（如 Industrial Craft、Thermal Series）。
- 魔法类：加入法杖、咒语、魔法生物（如 Botania、Ars Nouveau）。
- 生物类：加入更多动物和怪物（如 Alex's Mobs、Lycanites Mobs）。
- 冒险类：加入新维度、新地牢、新 Boss（如 Twilight Forest、The Aether）。
- 科技类：从手搓木棍到造火箭上天（如 Mekanism、Create）。

4. 视觉/辅助类——让画面更好看或信息更直观

- 高清字体 / 界面美化：让 UI 更精致。
- 血量显示：怪物头顶显示血条。
- 光影加载器：前面讲过的 Iris、OptiFine 本质上也是模组。

## 模组不能单独装，需要"加载器"

模组文件本身只是一堆代码，游戏不认识它，需要一个中间人来牵线搭桥。这个中间人就是模组加载器。

目前主流的有四个：

| 加载器 | 特点 |
|--------|------|
| Forge | 老牌模组加载器，有大量**旧版本模组**，社区庞大功能丰富但加载较重，**性能一般**，对新版本支持较慢，适合需要大量老牌模组或大型整合包的玩家 |
| Fabric | **轻量级、加载快、优化好**，新版本模组多，旧版支持有限。适合追求**高性能**、玩最新版本的玩家 |
| NeoForge | 较Forge更新速度更快，**性能更优秀**；较Fabric**功能更丰富**。适合玩高版本的模组 |
| Quilt | 是Fabric的一个社区分支，支持绝大多数Fabric模组 |

:::tip 小贴士
Forge 、NeoForge 和 Fabric 都不能随意混用。你装了 Forge，就只能装 Forge 版的模组；装了 Fabric，就只能装 Fabric 版的模组。下载模组时，一定要看清它写的是 "Forge" 、 "Fabric" 还是 "NeoForge"
:::

## 手机 Java 启动器能装模组吗？

能装，但要看你的手机性能和模组体量。

手机 Java 启动器（如 Fold Craft Launcher、PojavLauncher）本质上就是让你在手机上跑电脑版 Minecraft，所以电脑能装的模组，手机理论上都能装。但实际体验差距很大：

| 模组类型 | 手机表现 |
|---------|---------|
| **轻量优化模组**（Sodium、Lithium 等） | 推荐装，能提升帧数，对手机有帮助。 |
| **小功能模组**（小地图、JEI 等） | 可以装，影响不大。 |
| **大型内容模组**（工业、魔法、新维度） | 能吃下，但加载时间会变长，内存占用增加，低端机容易闪退。 |
| **大型整合包**（上百个模组打包） | 不建议。手机运存和 CPU 撑不住，大概率闪退或卡死。 |

:::tip 给手机玩家的建议
手机上玩模组，建议精挑细选，别贪多。选你最需要的（比如 Sodium + 小地图 + JEI），别一股脑塞几百个。另外，装模组前记得给启动器分配足够的内存（一般分配**占你手机总运存的70%左右**的运存），不然进都进不去。

部分模组不支持在手机上运行，主要是因为模组依赖的相关运行库没有迁移到手机上，这种情况下你只能放弃这个模组
:::

## 装模组前要知道的事

- 游戏版本要对上：1.20.1 的模组不能装到 1.19 里，1.21 的模组也不能装到 1.20 里。下载时看清版本号。
- 模组之间可能冲突：两个模组改了同一个东西，游戏就可能崩溃。这也是为什么"整合包"很流行——作者已经帮你排好雷了。
- 先备份存档：装新模组前，把原来的世界复制一份。万一模组把存档搞坏了，还有退路。
- 正版离线都能装：模组不区分正版还是离线，只要游戏能运行就能装。

## 模组和插件的区别

你可能还听过"插件"（Plugin），容易和模组搞混：

| 模组（Mod） | 插件（Plugin） |
|--|------------|---------------|
| **装在哪** | 装在你自己的游戏客户端里 | 装在服务器上 |
| **谁需要装** | 每个玩家都要装 | 玩家不用装，进服务器自动生效 |
| **能改什么** | 能改游戏底层，加新物品、新生物、新机制 | 只能改服务器层面的规则（如领地保护、经济系统、传送指令） |
| **典型例子** | 工业、小地图、暮色森林 | Essentials、WorldEdit、领地插件 |

:::note 注意
模组是"改游戏"，不是"改存档"。卸载模组后，之前用模组添加的物品可能会消失，但原版的内容不受影响。所以别在珍贵的存档里随便试来历不明的模组。
:::
