# CPU

> 注意：这里仅仅只介绍了手机CPU的概念

CPU 是中央处理器，相当于手机里的"总经理"——你点屏幕、打开 App、游戏里算伤害，这些逻辑任务都归它管。GPU 负责画图，NPU 负责 AI，CPU 则是什么都要管一点。

判断 CPU 强不强，主要看三个参数：

频率（GHz）：表示 CPU 每秒能运算多少次，数字越高单次计算越快。但频率高了发热和耗电也跟着涨，手机散热有限，长时间高负载会自动降速。

核心数：现在手机 CPU 基本都是 8 核，但核分三种——大核干重活（开游戏、加载大 App），中核干日常活（刷视频、聊天），小核干轻活（后台听歌、收消息）。这种搭配是为了省电：轻活交给小核，重活才唤醒大核。所以别只看"8 核"这个数字，大核是什么型号才关键。

架构：手机 CPU 都用 ARM 架构，越新的架构同频率下性能越强、越省电。比如新架构的 3.0 GHz 可能比老架构的 3.3 GHz 还快。

对普通用户来说，CPU 决定了手机卡不卡、能同时开多少 App、游戏流不流畅。核心数不是越多越好，架构新、大核强，日常使用体验才真的好。
