# GPU

> 注意：这里仅仅只介绍了手机GPU的概念
> 详细文章请前往[GPU](/docs/%E6%89%8B%E6%9C%BA%E5%B0%8F%E7%99%BD%E5%BF%85%E7%9C%8B/%E6%89%8B%E6%9C%BA%E9%85%8D%E7%BD%AE%E8%AE%B2%E8%A7%A3/gpu)文档查看

GPU 是图形处理器，相当于手机里的"美工部"——游戏画面、动画效果、视频播放、界面滑动，所有和"显示"有关的事都归它管。

它和 CPU 的分工很明确：CPU 像总经理，核心少但能力强，一件一件处理复杂任务；GPU 像一群画师，核心几百上千个，每个只会画简单的像素，但能同时开工，所以画图特别快。

手机 GPU 没有独立显卡，都集成在 SoC 里。

主流三家：Adreno（高通骁龙）、Mali（联发科天玑、华为麒麟）、Apple GPU（苹果自研）。

GPU 也有频率（单位 MHz 或 GHz），频率越高单时间处理的像素越多，画面越流畅。但频率高意味着发热大、耗电多，而且不同架构的 GPU 频率不能直接比较。

想知道 GPU 强不强，可以查「[手机 GPU 天梯图](https://www.socpk.com/gpu/)」或看 3DMark、GFXBench 等图形跑分。

对普通用户来说，玩游戏卡不卡，GPU 也是关键之一
