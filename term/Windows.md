# Windows

Windows 是微软公司开发的操作系统，也是目前世界上使用最广泛的电脑系统。你买的绝大多数台式机、笔记本，开机看到的那个有"开始菜单"的界面，基本就是 Windows。

[detail]

## Windows 长什么样？

Windows 最大的特点就是图形化界面——桌面上有图标，下面有任务栏，左下角有个"开始"按钮，用鼠标点来点去就能操作。不需要记命令，不需要敲代码，上手门槛很低，这也是它普及的最大原因。

## 常见的 Windows 版本

你可能听过这些名字，简单理一下它们的关系：

| 名字 | 大白话介绍 |
|------|-----------|
| **Windows 7** | 老经典，很多老电脑还在用，但微软已经停止维护了，不推荐新手再用。 |
| **Windows 10** | 目前保有量很大的版本，稳定、兼容性好，很多老软件都能跑。 |
| **Windows 11** | 微软主推的新版本，界面更现代，但对电脑硬件有一定要求，老电脑可能装不上。 |

:::tip 小贴士
"家庭版""专业版""企业版"等后缀是微软针对不同用户群体推出的功能版本，如果你懒得区分，Win10/Win11 无脑选专业版，Win7 无脑选旗舰版，功能最全、限制最少。
:::

## Windows 的优点

- 软件和游戏最多：你想用的软件、想玩的游戏，基本都有 Windows 版。
- 兼容性最好：各种打印机、网银、办公软件、老程序，Windows 支持得最完善。
- 上手简单：不用学命令，鼠标点点就能完成大部分操作。

## Windows 的"槽点"

- 更新有点烦：经常会自动下载更新。
- 自带广告/推广：开始菜单、锁屏、文件管理器里偶尔会冒出微软自家的推广。
- 需要注意安全：因为用的人多，病毒和流氓软件也最爱盯上 Windows，需要装杀毒软件或注意安全习惯。

## 额外补充

- 激活：正版 Windows 需要激活，没激活也能用，但桌面右下角会有水印，部分功能受限。品牌电脑一般预装好了，不用自己操心。
- 重要资料、下载的文件：别放桌面或 C 盘，系统出问题容易丢。
- 软件装哪？ 看 C 盘大小：

> C 盘 ≥ 150GB：中小型软件（浏览器、办公软件、聊天工具等）默认装 C 盘即可；大型软件（游戏、视频剪辑、设计软件）建议装 D 盘。

> C 盘 < 150GB：不管大小，软件都建议装 D 盘，不然 C 盘很容易爆满变红。

:::note 注意
Windows 和 Linux、macOS 不同，它的系统文件和软件安装位置相对固定（主要在 C 盘）。如果你从其他系统切换过来，或者在网上下载文件，注意这些习惯差异，能减少很多麻烦。
:::
