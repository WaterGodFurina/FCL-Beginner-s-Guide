警告内容​

警告

你必须开启版本隔离，否则当你安装多个版本（带模组之类）时，会出现不同版本用相同的模组，导致游戏崩溃。

注意内容​

注意

云控本身是中性的技术，用来推送更新、远程诊断是好事。但厂商怎么用、用在哪、是否告知用户，才是关键。买手机时可以查一下该厂商的云控口碑。

提示

模组的路径位于 /storage/emulated/0/FCL/.minecraft/versions/版本文件夹/mods， 光影的路径位于 ./minecraft/versions/版本文件夹/shaderpacks。

跳转按钮（Docusaurus 原生）​

Docusaurus 文档页面底部自动生成上一篇/下一篇导航按钮，无需手动添加。

如果你想在文档内部添加导航卡片，可以使用 DocCardList：

<DocCardList />

或者手动指定卡片：

<DocCard item={{
  type: 'link',
  label: '上一篇：账号登录',
  docId: 'account-login',
  href: '/docs/account-login'
}} />
