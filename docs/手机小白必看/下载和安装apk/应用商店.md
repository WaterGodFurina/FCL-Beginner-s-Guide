# 应用商店

## 什么是应用商店？

**应用商店**（App Store / 应用市场）是手机系统自带的、**专门用来下载和安装 App 的工具**。你打开它，搜 App 名字，点「安装」或「获取」，它就自动下载并装好，**不用你管后缀名、不用改路径、不用开权限**——一切自动。

打个比方：应用商店就像「超市」，App 是货架上的商品，你挑好放购物车（下载），结账后商品就送到家（安装）。整个过程**安全、方便、自动**。

## 各家应用商店

不同手机厂商自带不同的应用商店：

| 厂商 | 应用商店名字 |
|------|-------------|
| 华为 | [华为应用市场（AppGallery）](https://consumer.huawei.com/cn/mobileservices/appgallery/) |
| 小米 | [小米应用商店](https://app.mi.com/) |
| OPPO | [软件商店](https://store.oppomobile.com/) |
| vivo | [应用商店](https://h5.appstore.vivo.com.cn/#/) |
| 三星 | [Galaxy Store](https://galaxystore.samsung.com/) |
| 苹果 | [App Store（iOS）](https://www.apple.com/app-store/) |
| 谷歌 | [Google Play](https://play.google.com/)（国内手机通常没有） |

除了系统自带的，还有第三方应用商店：

- [**应用宝**](https://sj.qq.com/)（腾讯）
- [**360 手机助手**](https://zs.360.cn/)
- [**豌豆荚**](https://www.wandoujia.com/)（老牌，现在用得少）
- [**APKPure**](https://apkpure.net/) / [**APKMirror**](https://www.apkmirror.com/)（国外的，能下 Google Play 上的 App）

## 应用商店装 App 的流程

1. **打开应用商店**。
2. **搜索 App 名字**（比如「微信」）。
3. **点「安装」或「获取」**。
4. **自动下载 + 自动安装**，不用你操作。
5. 装完桌面出现图标，点开就能用。

整个过程**不用改后缀名、不用开权限、不用找路径**，应用商店全包了。这就是为什么应用商店是**最推荐**的安装方式。

## 应用商店的好处

### 1. 安全

应用商店里的 App 都经过**审核**（虽然各家审核严格程度不同），**比从网页下载的 APK 安全得多**。商店会检查：

- 有没有病毒
- 有没有恶意权限
- 有没有盗版
- 有没有违法内容

### 2. 自动更新

App 有新版时，应用商店会自动更新，**不用你手动下载新版 APK 再装一遍**。

### 3. 自动适配

应用商店会根据你手机的 CPU 架构、Android 版本、屏幕分辨率，**自动下发合适的 APK**。比如你手机是 arm64，就下发 arm64 版本，不下发 x86 版本，省空间。

### 4. 一键卸载

在应用商店里能一键卸载 App，**清理残留文件**，比手动卸载干净。

## 应用商店的局限

### 1. 上架慢

开发者要把 App 提交审核，**审核要几小时到几天**。所以应用商店里的版本**可能比官网最新版慢**。

### 2. 部分应用不上架

有些 App 因为政策、版权、内容等原因，**不上架应用商店**，只能从官网下载。比如：

- 一些海外 App（国内应用商店没有）
- 一些破解版、修改版（违反商店政策）
- 一些小众工具

### 3. 厂商商店可能限制

有些厂商的应用商店会**优先推荐自家应用**，或者**限制安装非商店来源的 App**（比如华为纯净模式、小米的「安全守护」）。这是厂商的「安全策略」，但也让用户不自由。

### 4. 国内没有 Google Play

国内手机基本没有 Google Play，所以**Google 系应用（Gmail、YouTube、Maps）和很多海外 App 装不了**。要装这些，得用 APKPure、APKMirror 等第三方商店，或者从官网下 APK。

## 应用商店 vs 手动装 APK

| 对比项 | 应用商店 | 手动装 APK |
|--------|----------|------------|
| 方便程度 | 一键安装 | 要找路径、改后缀、开权限 |
| 安全性 | 高（有审核） | 看来源，可能不安全 |
| 更新 | 自动 | 手动 |
| 最新版 | 可能慢 | 能装最新 |
| 海外 App | 国内商店没有 | 能装 |
| 破解版 | 没有 | 能装（但违规） |

**建议**：能用应用商店就用应用商店；商店没有的，再从官网或可信渠道下 APK 手动装。

## 小结

- **应用商店 = 系统自带的 App 下载安装工具**，最安全最方便。
- 各家厂商有自家商店：华为应用市场、小米应用商店、OPPO 软件商店等。
- 流程：搜索 → 安装 → 自动下载安装。
- 好处：安全、自动更新、自动适配、一键卸载。
- 局限：上架慢、部分应用没有、国内没 Google Play。
- **能用商店就别手动装 APK**。
