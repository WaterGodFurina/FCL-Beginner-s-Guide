# 如何查看自己的手机配置
有些小白在面对大佬的回答时可能有些疑惑：怎么查看自己的手机配置？

## 手机设置

不同品牌路径略有差异：

- 小米/红米：设置 → 我的设备
- 华为/荣耀：设置 → 关于手机
- OPPO/realme/一加：设置 → 关于手机 → 版本信息
- vivo/iQOO：设置 → 系统管理 → 关于手机
- 三星：设置 → 关于手机 → 信息
- Google Pixel：设置 → 关于手机
- 苹果iOS：设置 → 通用 → 关于本机
## 浏览器查询

在网上搜索自己的手机型号，基本上可以查到自己的手机配置

## 第三方软件查询
| App 名称 | 平台 | 特点 |
|---|---|---|
| AIDA64 | Android / iOS | 最全面的硬件检测，显示 SoC、GPU、传感器等所有细节 |
| DevCheck | Android | 界面美观，实时显示 CPU/GPU 频率、温度 |
| Device Info HW | Android | 专为 Android 设计，显示驱动版本等底层信息 |
| Geekbench | Android / iOS | 跑分 + 硬件信息 |
| 安兔兔评测 | Android / iOS | 国内常用，跑分 + 硬件信息 |
| 3DMark | Android / iOS | GPU 性能测试 + 部分硬件信息 |
