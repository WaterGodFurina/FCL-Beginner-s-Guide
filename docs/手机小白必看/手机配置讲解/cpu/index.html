CPU 介绍

CPU 是什么？​

CPU 全称 Central Processing Unit，中文叫「中央处理器」。它是 SoC 里负责「算账」的部分——你点一下屏幕、打开一个 App、游戏里算一下怪物掉多少血，这些逻辑任务都归 CPU 管。

打个比方：如果手机是一家公司，CPU 就是公司的「总经理」，负责下指令、做决策、协调各部门。GPU 是「美工部」，专门画图；NPU 是「AI 顾问」，专门算 AI。CPU 什么都要管一点，所以它的「通用计算能力」最重要。

CPU 性能好不好，主要看三个参数：频率、核心数、架构。

一、频率（主频）​

频率也叫主频，单位是 GHz（吉赫兹），表示 CPU 每秒钟能「跳」多少次。比如 3.0 GHz，意思是这块 CPU 每秒能做 30 亿次基本操作（粗略理解）。

频率越高，单次计算越快。你可以把它想象成「厨师切菜的速度」——切得越快，菜出得越快。

但要注意：

  • 频率不是越高越好。频率高了，发热和耗电也跟着涨。手机散热有限，长时间高频会「降频」（自动降速来降温）。
  • 不同架构的频率不能直接比。比如苹果 A17 的 3.7 GHz 和骁龙 8 Gen3 的 3.3 GHz，因为架构不同，不能简单说「苹果更快」。但同架构同代产品，频率高的确实更快。

现在手机 CPU 的大核频率一般在 2.8 GHz ~ 4.3 GHz 之间，旗舰 SoC 大核频率能到 3.3 GHz 以上。

二、核心数​

核心数就是 CPU 里有多少个「干活的小工」。现在的手机 CPU 几乎都是多核的，常见的是 8 核（octa-core）。

核心分两种：

  • 大核（Prime / Cortex-X）：性能强、耗电多，干重活（开 App、加载游戏）。
  • 中核（Cortex-A7xx）：性能中等、耗电中等，干日常活（刷视频、聊天）。
  • 小核（Cortex-A5xx）：性能弱、省电，干轻活（后台听歌、收消息）。

这种「大核+中核+小核」的设计叫 big.LITTLE 架构，目的是省电：轻活交给小核，重活才唤醒大核。

常见的 8 核组合比如「1 大核 + 3 中核 + 4 小核」（1+3+4）或「1 大核 + 5 中核 + 2 小核」（1+5+2）。看核心数时，别只看「8 核」这个数字，要看大核是什么型号——大核才是性能担当。

小提醒

有些广告会宣传「十核处理器」来显得很厉害，但如果十个核都是小核，性能可能还不如一个 1+3+4 的 8 核。核心数 ≠ 性能，架构和单核性能更重要。

三、架构（指令集架构）​

架构这里特指 指令集架构（ISA，Instruction Set Architecture），可以理解为 CPU 能听懂的「语言」。

手机 CPU 几乎都用 ARM 架构（英国 ARM 公司设计）。ARM 架构的特点是省电，所以手机、平板、手表都用它。电脑上的 CPU（Intel、AMD）用的是 x86 架构，性能强但费电。

ARM 架构下又分具体型号，比如：

  • Cortex-X4 / X5：超大核，性能最强。
  • Cortex-A720 / A725：中核。
  • Cortex-A520：小核，省电。

苹果 A 系列虽然也是 ARM 架构，但苹果自己设计核心（不直接用 ARM 公版），所以性能比同代 ARM 公版强不少。

架构越新，同频率下性能越强、能效越好。这就是为什么「骁龙 8 Gen3 的 3.3 GHz」可能比「骁龙 8 Gen1 的 3.0 GHz」快很多——架构升级带来的提升，有时比频率提升还明显。

怎么看 CPU 参数？​

打开手机「设置 → 关于手机」，能看到处理器型号。比如：

处理器：八核处理器
CPU：1×3.3GHz + 3×3.2GHz + 4×2.2GHz

这段话的意思是：8 核 CPU，1 个 3.3 GHz 大核 + 3 个 3.2 GHz 中核 + 4 个 2.2 GHz 小核。结合前面讲的，你就能大致判断这块 CPU 的水平了。

小结​

  • CPU = SoC 里的「总经理」，负责通用计算。
  • 频率：越高越快，但发热也大，单位 GHz。
  • 核心数：现在都是多核（8 核常见），分大核/中核/小核，别只看核数。
  • 架构：手机用 ARM 架构，越新越强越省电。
