GPU 介绍

GPU 是什么？​

GPU 全称 Graphics Processing Unit，中文叫「图形处理器」。它是 SoC 里专门负责「画图」的部分——游戏画面、动画、视频解码、UI滑动，凡是和「显示」有关的，基本都归 GPU 管。

打个比方：CPU 是「总经理」，什么都要管一点，但每件事都得一件一件做；GPU 是「美工部」，里面有几百上千个小工，每个小工只会画一个像素，但它们能同时画，所以画图特别快。

这就是 GPU 和 CPU 的根本区别：

CPUGPU
核心数少（4–8 个）多（几百到几千个）
每个核心的能力强，能干复杂活弱，只会干简单活
擅长逻辑、判断、串行任务并行计算、画图、矩阵运算
比喻总经理美工部一群画师

手机 GPU 的主流厂商​

手机 GPU 都是集成在 SoC 里的，没有独立显卡。主流的有：

  • Adreno：高通骁龙自带，比如 Adreno 750、Adreno 740。
  • Mali：ARM 公司设计，联发科天玑、华为麒麟常用，比如 Mali-G715、Mali-G720。
  • PowerVR：Imagination 公司设计，早年苹果 A 系列用过，现在主要是一些中低端 SoC。
  • Apple GPU：苹果自研，A 系列自带，比如 A17 Pro 的 6 核 GPU。

GPU 频率意味着什么？​

和 CPU 一样，GPU 也有频率，单位也是 MHz 或 GHz。比如「Adreno 750 @ 770 MHz」意思是这块 GPU 跑在 770 MHz。

频率越高，单时间能处理的像素越多，画面越流畅。但同样地：

  • 频率高 = 发热大 = 耗电多。
  • 不同架构的 GPU 频率不能直接比。比如 Mali-G720 @ 800 MHz 和 Adreno 750 @ 770 MHz，不能简单说前者快。

现在旗舰手机 GPU 频率一般在 700 MHz ~ 1000 MHz 之间。

GPU 性能怎么看？​

看 GPU 性能，最直观是看图形跑分：

  • 3DMark：国际通用的图形跑分，有专门针对手机的 Wildlife 场景。
  • GFXBench：专门测游戏画面帧率。
  • 安兔兔 GPU 子分：国内常用。

也可以查「手机 GPU 天梯图」，能看到各款 GPU 的大致排名。

GPU 和游戏的关系​

玩游戏卡不卡，GPU 是关键。但要注意：

  1. GPU 决定上限，CPU 决定下限。GPU 强，画面能开高画质；但如果 CPU 太弱，游戏逻辑算不过来，照样卡。
  2. 散热决定能跑多久。GPU 满载发热大，手机散热差的话，玩 10 分钟就降频，帧率从 60 掉到 30。
  3. 分辨率影响 GPU 负载。2K 屏比 1080P 屏让 GPU 多干一倍的活，所以同款 SoC 在 2K 屏上玩游戏更吃力。

小结​

  • GPU = SoC 里的「美工部」，专管画图，核心多、擅长并行。
  • 主流手机 GPU：Adreno（高通）、Mali（联发科/麒麟）、Apple GPU。
  • GPU 频率越高画图越快，但发热也大，不同架构不能直接比。
  • 看性能查「GPU 天梯图」或 3DMark 跑分。
