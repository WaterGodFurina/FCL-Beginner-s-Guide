# 安卓版本

## Android 是什么？

**Android**（中文叫「**安卓**」）是 Google（谷歌）开发的**手机操作系统**，和苹果的 iOS 是手机界两大主流系统。

操作系统是什么？打个比方：手机硬件（SoC、屏幕、电池）是「身体」，操作系统是「大脑」。没有操作系统，手机就是块砖；有了操作系统，你才能点屏幕、装 App、打电话。

Android 的特点：

- **开源**：任何人都能拿到源代码自己改（这就是为什么小米的 MIUI、OPPO 的 ColorOS、华为的 EMUI 都基于 Android）。
- **基于 Linux 内核**：Linux 是一种开源的操作系统内核，Android 借用了它的底层。
- **应用是 APK 格式**：Android 的 App 都是 `.apk` 文件，电脑上的软件是 `.exe`，互不通用。

## Android 版本号怎么排？

Android 从 2008 年的 1.0 开始，到现在已经到 Android 15（2024 年发布）。每个大版本都有一个**甜点代号**，按字母顺序排：

| 版本 | 代号 | 发布年份 |
|------|------|----------|
| Android 9 | Pie（派） | 2018 |
| Android 10 | 没有甜点名（Quince Tart） | 2019 |
| Android 11 | Red Velvet Cake（红丝绒蛋糕） | 2020 |
| Android 12 | Snow Cone（雪糕） | 2021 |
| Android 13 | Tiramisu（提拉米苏） | 2022 |
| Android 14 | Upside Down Cake（倒置蛋糕） | 2023 |
| Android 15 | Vanilla Ice Cream（香草冰淇淋） | 2024 |

从 Android 10 开始，Google 不再公开甜点代号，但内部还在用。

除了数字版本，Android 还有**API 级别**（API Level），是给开发者用的。比如 Android 14 对应 API 34，Android 15 对应 API 35。你装 App 时如果提示「此应用需要 Android 11 以上」，就是看 API 级别。

## 各版本有什么区别？

对普通用户来说，最关心的是**新版本带来了什么新功能、新限制**。下面挑几个重要的讲：

### Android 11（2020）

- **`Android/data` 文件夹访问受限**：普通文件管理器打不开了，需要用系统自带的「文件管理」或支持 SAF 的工具。这是后面会专门讲的重点。
- 聊天气泡（像微信那种悬浮窗）。
- 一次性权限（可以只授权一次）。

### Android 12（2021）

- **Material You 设计**：系统界面大改，颜色跟随壁纸。
- 隐私仪表盘：能看到哪些 App 在用相机、麦克风。
- 大幅限制后台启动。

### Android 13（2022）

- **媒体权限细分**：以前一个「存储」权限就能读所有照片，现在要分别申请「照片」「视频」「音频」权限。
- **照片选择器**：App 只能选你挑的照片，不能一次读全部。
- 通知权限要单独申请。

### Android 14（2023）

- **部分照片访问**：可以只给 App 看几张照片，而不是全部。
- 锁屏自定义。
- 字体可放大到 200%。

### Android 15（2024）

- **私密空间**：把敏感 App 藏起来，需要指纹才能进。
- 改进的通知。
- 更严格的后台限制。
- 折叠屏、平板体验优化。

## 为什么 Android 版本很重要？

很多 App 会要求**最低 Android 版本**。比如某个新游戏可能要求 Android 11 以上，你手机是 Android 10 就装不上。原因：

1. 新 Android 加了新 [API](https://zhuanlan.zhihu.com/p/347125981)，App 用了新 API 就跑不了旧系统。
2. 新 Android 加了新限制，旧 App 在新系统上可能跑不了（比如旧版文件管理器在 Android 11 上打不开 data）。

所以买手机时，**Android 版本越新越好**，能用更久、兼容更多新 App。

## 怎么看自己手机的 Android 版本？

打开「**设置 → 关于手机 → Android 版本**」就能看到。比如显示「Android 版本 14」，就是 Android 14。

厂商的系统（MIUI、ColorOS、HyperOS 等）有自己的版本号，但底层 Android 版本才是关键。

## 小结

- **Android = Google 开发的开源手机操作系统**，基于 Linux 内核。
- 版本从 1.0 到 15，每个版本有甜点代号。
- 新版本带来新功能，也带来新限制（比如 Android 11 限制 data 访问）。
- 看 Android 版本：设置 → 关于手机 → Android 版本。
