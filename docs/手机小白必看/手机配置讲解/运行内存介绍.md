# 运行内存介绍

## 运行内存是什么？

**运行内存**，英文 **RAM**（Random Access Memory，**随机存取存储器**），是手机里**临时存数据**的地方。

打个比方：手机就像一个办公桌，**运行内存就是桌面**。你打开微信、抖音、游戏，这些 App 的数据就「摊」在桌面上，CPU 随时能拿到。桌面越大，能同时摊开的 App 越多，手机越不容易卡。

运行内存的特点：

- **速度快**：读写比内部存储快几十倍。
- **临时性**：**断电数据就没了**。你关机重启后，所有 App 都得重新打开，就是因为运行内存被清空了。
- **容量小**：现在手机运行内存一般是 6GB、8GB、12GB、16GB，比内部存储（128GB、256GB）小很多。

## 运行内存为什么「断电即失」？

这是 RAM 的物理特性决定的。手机用的 RAM 叫 **DRAM**（动态随机存取存储器），它存数据靠的是**电容里的电荷**——有电荷是 1，没电荷是 0。

电容里的电荷会**慢慢漏掉**，所以 RAM 需要不断「刷新」（refresh）来维持数据，这个过程叫**动态刷新**。一旦断电，电容里的电荷几毫秒内就漏光了，数据自然全没。

这就是为什么：

- 关机重启后，所有 App 都要重新打开。
- 手机突然没电关机，正在编辑没保存的文档可能丢。
- 蓝屏、卡死后强制重启，没保存的工作会丢。

:::note 名词解释
**电容**：一个能存少量电荷的电子元件，像个小水杯。RAM 里每个存数据的「格子」就是一个电容。
:::

## 运行内存大小有什么影响？

运行内存大小，主要影响**多任务能力**：

| 运行内存 | 大概能同时开几个 App |
|----------|---------------------|
| 4GB | 微信 + 抖音 + 浏览器，再多就杀后台 |
| 6GB | 微信 + 抖音 + 游戏 + 浏览器 |
| 8GB | 主流够用，能开 6–8 个常用 App |
| 12GB | 流畅，游戏 + 多个 App 都不杀 |
| 16GB | 旗舰级，重度多任务无压力 |
| 24GB | 极端配置，目前顶配 |

「杀后台」就是系统为了腾出运行内存，把后台 App 关掉。你切回微信发现要重新加载，就是被杀后台了。

## 运行内存的「类型」

手机 RAM 有不同代，越新越快越省电：

- **LPDDR4X**：老机型，2018 年前后。
- **LPDDR5**：2020 年前后主流。
- **LPDDR5X**：2022 年起旗舰标配，比 LPDDR5 快 30%、省电 20%。
- **LPDDR5T**：2023 年起顶级旗舰，比 LPDDR5X 再快 13%。

**LPDDR** 全称 **Low Power Double Data Rate**，是专为移动设备设计的低功耗内存标准。后面的 X、T 是增强版。

## 运行内存和「内存融合」是什么？

有些厂商宣传「**内存融合**」「**内存扩展**」，比如 8GB 运行内存 + 4GB 扩展 = 12GB。这其实是**从内部存储里划一部分当虚拟内存用**，类似电脑的虚拟内存（pagefile）。

好处：能多开几个 App。坏处：内部存储比 RAM 慢得多，划出来的虚拟内存速度也慢，**不能真正替代物理 RAM**。所以「8+4」实际体验不如真 12GB。

## 怎么看运行内存？

打开「**设置 → 关于手机 → 运行内存**」或「**设置 → 运行内存和存储**」能看到。比如「8GB + 256GB」，前面 8GB 就是运行内存，后面 256GB 是内部存储。

## 小结

- **运行内存（RAM）= 办公桌桌面**，临时放 App 数据，**断电即失**。
- 容量越大，能同时开的 App 越多，越不容易杀后台。
- 类型越新越快：LPDDR5X > LPDDR5 > LPDDR4X。
- 「内存融合」是从存储划虚拟内存，速度慢，不能替代物理 RAM。
