# 其他部分

SoC 里除了 CPU 和 GPU，还有一堆「专才」——它们各自只干一件事，但干得又快又省电。这一节简单介绍几个常见的。

## NPU（神经网络处理器）

**NPU** 全称 **Neural Processing Unit**，中文叫「**神经网络处理器**」或「**AI 处理器**」。它专门用来跑 **[AI 模型](https://zhuanlan.zhihu.com/p/24645316686)**（神经网络），比如：

- 相机的「AI 美颜」「AI 场景识别」
- 语音助手的语音识别
- 相册里的人脸识别、智能分类
- 文字输入法的智能预测
- 现在的「端侧大模型」（手机本地跑 AI 对话）

为什么 AI 要单独搞个 NPU？因为 AI 计算主要是**[矩阵乘法](https://www.luogu.com.cn/article/2axvf8sn)**，CPU 不擅长，GPU 擅长但费电。NPU 专门为矩阵运算设计，**又快又省电**。

NPU 性能常用 **TOPS**（每秒万亿次操作）衡量，比如「26 TOPS」。数字越大，AI 算力越强。现在旗舰 SoC 的 NPU 算力普遍在 **20–50 TOPS**。

:::note 名词解释
**AI 模型 / 神经网络**：可以理解为一个「经过大量数据训练后，能自己判断结果」的程序。比如喂它几百万张猫狗照片，它就能学会分辨猫和狗。这种程序跑起来要算海量乘法，所以需要 NPU。
:::

## ISP（图像信号处理器）

**ISP** 全称 **Image Signal Processor**，中文叫「**图像信号处理器**」。它专门处理**相机拍到的原始图像**。

手机拍照流程大致是：镜头 → [传感器（CMOS）](https://baike.baidu.com/item/%E4%BC%A0%E6%84%9F%E5%99%A8/26757)→ ISP → 最终照片。ISP 负责：

- 把传感器输出的原始数据（Bayer 数据）转成彩色照片
- 降噪（暗光下抹掉噪点）
- 白平衡（让白色看起来是白色）
- HDR（高动态范围，亮处不过曝、暗处能看清）
- 美颜、滤镜、人像虚化

ISP 好不好，直接决定**拍照画质**。所以同款传感器（比如 IMX989），不同厂商调出来的效果差很多，很大程度就是 ISP 和算法的差异。

## DSP（数字信号处理器）

**DSP** 全称 **Digital Signal Processor**，中文叫「**数字信号处理器**」。它专门处理**连续的信号**，比如：

- 音频处理（降噪、回声消除、杜比全景声）
- 语音通话处理
- 一些传感器数据（陀螺仪、加速度计）的实时处理

DSP 比 CPU 省电，比 NPU 通用，介于两者之间。日常用得不多，但打电话清楚、录音干净，DSP 功不可没。

## 基带（Modem）

**基带**也叫 **Modem**（调制解调器），是 SoC 里管**手机信号**的部分。它负责：

- 4G / 5G 蜂窝网络连接
- 通话、短信
- 与基站的信号收发

基带好不好，决定**信号强弱、网速快慢、5G 支不支持**。比如骁龙 8 Gen3 集成的基带支持 [5G 毫米波](https://baike.baidu.com/item/5G%E6%AF%AB%E7%B1%B3%E6%B3%A2/58513934)和 [Sub-6GHz](https://baike.baidu.com/item/Sub-6GHz%E9%A2%91%E6%AE%B5/67192902)，下载峰值能到 10 Gbps（理论值）。

苹果以前外挂高通基带，后来想自研但翻车，又换回高通。基带技术门槛很高，全球能做的不多。

## 内存控制器（Memory Controller）

**内存控制器**是 SoC 和运行内存（RAM）之间的「桥梁」，负责数据搬运。它决定了：

- 内存频率能跑多高（比如 LPDDR5X-8533）
- 内存带宽多大（影响 GPU 性能发挥）

这个部件你日常感知不到，但它直接影响整机性能。

## 安全模块（Secure Processor / TEE）

现代 SoC 里还有专门的安全模块，比如高通的 **[SPU](/docs/technical-term/SPU)**、ARM 的 **[TrustZone](/docs/technical-term/TrustZone)**。它们负责：

- 指纹、人脸数据的安全存储
- 支付宝/微信支付的密钥
- 全盘加密的密钥

这部分和主 CPU 隔离，即使系统被黑，密钥也拿不走。

## 小结

| 部件 | 干什么 | 你能感知到 |
|------|--------|------------|
| NPU | AI 计算 | 相机 AI、语音助手、端侧大模型 |
| ISP | 相机图像处理 | 拍照画质 |
| DSP | 信号处理 | 通话清楚、录音干净 |
| 基带 | 蜂窝网络 | 信号强弱、5G |
| 内存控制器 | 数据搬运 | 整机流畅度 |
| 安全模块 | 密钥存储 | 指纹、支付安全 |

SoC 之所以叫「片上系统」，就是因为这些部件**全在一块芯片上**，各司其职又互相配合。
