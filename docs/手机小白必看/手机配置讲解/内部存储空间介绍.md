# 内部存储空间介绍

## 内部存储空间是什么？

**内部存储空间**，英文常叫 **ROM**（虽然严格说不准确，后面会纠错），是手机里**永久存数据**的地方。

继续用办公桌的比方：**内部存储就是抽屉/柜子**。你装的所有 App、拍的所有照片、下载的所有文件，都存在抽屉里。抽屉越大，能装的东西越多。**关机断电，抽屉里的东西还在**——这就是它和运行内存的根本区别。

内部存储的特点：

- **永久性**：断电数据不丢。
- **容量大**：现在手机一般 128GB、256GB、512GB、1TB。
- **速度慢**：比运行内存慢几十倍，但比 U 盘、SD 卡快。

## 内部存储为什么「断电还在」？

手机内部存储用的是 **闪存（Flash Memory）**，具体是 **NAND 闪存**。闪存存数据靠的是**浮栅晶体管**——把电荷「关」在一个绝缘的小盒子里，电荷跑不出去，所以断电数据还在。

和 RAM 的电容不同，闪存的电荷被绝缘层包住，**不会漏掉**，所以能长期保存。但写入时要先「擦除」再写，擦写次数有限（一般几万到几十万次），所以闪存有寿命。

:::note 名词解释
**闪存（Flash）**：一种断电不丢数据的存储芯片，U 盘、SD 卡、手机存储、固态硬盘（SSD）都用它。
:::

## 内部存储的「类型」

手机闪存主要有两种标准：

### eMMC（embedded Multi Media Card）

- 老技术，2018 年前的中低端机常用。
- 速度慢，类似电脑的机械硬盘。
- 现在基本被淘汰，只有百元机还在用。

### UFS（Universal Flash Storage）

- 新技术，2017 年起普及。
- 速度比 eMMC 快几倍，类似电脑的固态硬盘。
- 分代：UFS 2.1 → UFS 3.0 → UFS 3.1 → UFS 4.0。
- **UFS 4.0** 是 2022 年起的旗舰标配，读写速度能到 4 GB/s 以上。

UFS 比 eMMC 快的好处：

- 装大游戏（比如原神 20GB）快很多。
- 拍照存照片快，连拍不卡。
- 打开 App 快。

## 内部存储大小有什么影响？

| 容量 | 够用人群 |
|------|----------|
| 64GB | 只装几个常用 App、不拍照的人，已经很紧张 |
| 128GB | 轻度用户，能装 30–50 个 App + 少量照片 |
| 256GB | 主流够用，能装 100+ App + 几千张照片 |
| 512GB | 重度用户，存大量照片视频、玩多个大游戏 |
| 1TB | 极端需求，存 4K 视频、离线地图、大量音乐 |

注意：**系统本身要占 20–30GB**，所以 128GB 实际可用大概 100GB 左右。

## 内部存储和「读写速度」

存储不只是看容量，还要看**读写速度**。读取速度影响打开 App 快不快，写入速度影响拍照存照片快不快。

- **顺序读写**：连续大文件（视频、大游戏）的速度。
- **随机读写**：零碎小文件（系统文件、App 配置）的速度，**这个更影响日常流畅度**。

UFS 4.0 的顺序读取能到 4200 MB/s，eMMC 5.1 只有 250 MB/s，差十几倍。所以同款 SoC，UFS 4.0 的手机打开 App 明显比 eMMC 快。

## 怎么看内部存储？

打开「**设置 → 关于手机 → 存储空间**」或「**设置 → 运行内存和存储**」能看到。比如「8GB + 256GB」，256GB 就是内部存储。

## 小结

- **内部存储 = 抽屉/柜子**，永久存数据，**断电还在**。
- 用的是闪存（NAND Flash），靠绝缘层锁住电荷。
- 类型：UFS 4.0 > UFS 3.1 > UFS 2.1 > eMMC（已淘汰）。
- 容量看需求，128GB 起步，256GB 主流，512GB+ 重度用。
