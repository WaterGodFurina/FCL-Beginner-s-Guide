# 后缀名列举（主要）

## 什么是文件后缀名？

**文件后缀名**（也叫扩展名）就是文件名最后那个「**.xxx**」的部分。比如 `照片.jpg`，`.jpg` 就是后缀名；`游戏.apk`，`.apk` 就是后缀名。

后缀名的作用是**告诉系统和用户「这是个什么文件」**。系统看到 `.apk` 就知道要用安装器打开，看到 `.jpg` 就知道要用图片查看器打开。

:::note 注意
后缀名**只是个标签**，改后缀名不会改变文件内容。把 `照片.jpg` 改成 `照片.txt`，文件还是图片，只是系统会用记事本去打开它，显示一堆乱码。所以**改后缀名 ≠ 转换格式**。
:::

## 手机和电脑常见后缀名一览

下面按类别列举你常会遇到的后缀名。

### 一、安装包类

| 后缀名 | 是什么 | 用在哪 |
|--------|--------|--------|
| `.apk` | Android 安装包 | 安卓手机 |
| `.apks` | 拆分版 APK（多个 split） | 安卓手机 |
| `.xapk` | 带资源的 APK 包 | 安卓手机（APKPure 用） |
| `.aab` | Android App Bundle（开发者上传格式） | Google Play |
| `.exe` | Windows 可执行文件 | Windows 电脑 |
| `.msi` | Windows 安装包 | Windows 电脑 |
| `.deb` | Debian 系安装包 | Ubuntu、Debian 等 Linux |
| `.rpm` | Red Hat 系安装包 | Fedora、CentOS 等 Linux |
| `.dmg` | macOS 磁盘映像 | 苹果电脑 |
| `.ipa` | iOS 安装包 | iPhone（需越狱或签名） |

### 二、压缩包类

| 后缀名 | 是什么 | 特点 |
|--------|--------|------|
| `.zip` | 最常见压缩包 | 兼容性最好 |
| `.7z` | 7-Zip 压缩包 | 压缩率高 |
| `.rar` | WinRAR 压缩包 | 老牌，Windows 常见 |
| `.tar` | Linux 归档文件 | 只打包不压缩 |
| `.tar.gz` / `.tgz` | tar + gzip 压缩 | Linux 常见 |
| `.tar.xz` | tar + xz 压缩 | 压缩率最高 |
| `.gz` | gzip 单文件压缩 | Linux 常见 |
| `.bz2` | bzip2 单文件压缩 | Linux 常见 |

### 三、文本类

| 后缀名 | 是什么 |
|--------|--------|
| `.txt` | 纯文本文件 |
| `.md` | Markdown 文本（带格式） |
| `.log` | 日志文件（纯文本） |
| `.json` | JSON 数据文件（结构化文本） |
| `.xml` | XML 数据文件 |
| `.csv` | 逗号分隔数据表 |
| `.html` / `.htm` | 网页文件 |
| `.ini` / `.conf` | 配置文件 |

### 四、代码和库类

| 后缀名 | 是什么 |
|--------|--------|
| `.java` | Java 源代码 |
| `.class` | Java 编译后的字节码 |
| `.jar` | Java 归档包（多个 .class 打包） |
| `.py` | Python 源代码 |
| `.c` / `.cpp` | C / C++ 源代码 |
| `.so` | Linux/Android 共享库（类似 Windows 的 .dll） |
| `.dll` | Windows 动态链接库 |
| `.a` | Linux 静态库 |
| `.lib` | Windows 静态库 |
| `.sh` | Shell 脚本 |
| `.bat` | Windows 批处理脚本 |

## 常见后缀名详解

### `.apk`

Android 安装包，安卓手机装 App 用的。本质是个 ZIP 压缩包，里面装着 App 的代码、资源、签名。后面会专门讲怎么装。

### `.zip` / `.7z` / `.tar` / `.rar`

都是压缩包，区别在压缩算法和软件：

- `.zip`：最通用，所有系统都能解。
- `.7z`：7-Zip 软件的格式，压缩率高，但需要 7-Zip / Bandizip / MT管理器 等软件来解压。
- `.tar`：Linux 上常用，**只打包不压缩**，常配合 `.gz` 用成 `.tar.gz`。
- `.rar`：WinRAR 的格式，Windows 常见，手机解压需要 MT管理器 等支持 RAR 的工具。

### `.txt` / `.log`

都是纯文本：

- `.txt`：普通文本，记事本能开。
- `.log`：日志文件，记录程序运行信息，也是纯文本，记事本也能开。

### `.jar`

Java 归档包，里面装着 Java 编译后的字节码（`.class` 文件）。Minecraft Java 版的 mod 很多是 `.jar` 文件。

### `.so`

Linux/Android 的**共享库**（Shared Object），相当于 Windows 的 `.dll`。Minecraft Java版移植到手机时，关键就是把电脑上的 `.so` 文件重新编译成手机能跑的版本。后面会专门讲。

### `.exe`

Windows 可执行文件，**只能在 Windows 上跑**。手机直接打不开（除非用[模拟器](https://blog.csdn.net/weixin_42576804/article/details/117664604)）。

### `.deb` / `.rpm`

Linux 两大流派的安装包：

- `.deb`：Debian 系（Ubuntu、Debian、Linux Mint）用。
- `.rpm`：Red Hat 系（Fedora、CentOS、openSUSE）用。

### `.mcpack`

Minecraft 资源包/行为包格式，由 Mojang 定义，本质上是**重命名的 ZIP 压缩包**。用于打包 Minecraft 基岩版（Bedrock Edition）的材质、模型、音效、行为脚本等资源，可通过游戏内直接导入。

## 怎么看后缀名？

- **手机**：默认可能不显示后缀名，需要在文件管理器设置里开「显示后缀名」。
- **Windows**：默认也不显示，要在「文件夹选项 → 查看」里取消勾选「隐藏已知文件类型的扩展名」。
- **Linux/Mac**：默认显示。

## 小结

- 后缀名 = 文件名最后的 `.xxx`，告诉系统这是什么文件。
- 常见类别：安装包、压缩包、文本、代码库、媒体、镜像。
- 改后缀名不改变文件内容，只是改了「标签」。
- 手机和 Windows 默认隐藏后缀名，要去设置里开显示。
