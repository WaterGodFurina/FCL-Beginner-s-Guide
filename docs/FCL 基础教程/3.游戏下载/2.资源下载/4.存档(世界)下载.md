# 存档下载&安装

>**前情提要**：存档下载不同于其它资源的下载，其它资源只需下载就行了，而存档的安装步骤有些繁琐。
由于存档是一个文件夹，只能将其**打包为压缩包**的形式才能通过互联网传播，所以下载下来的存档**需要解压**。

因为过于繁琐所以将本文分为**下载**和**安装**两个部分来讲

## 存档的下载

### FCL内置下载

1. 打开FCL下载页面，点击中间上方第五个“**世界**”
2. 筛选或搜索你想要的世界。中间的左侧可以筛选游戏版本、类型等，也可以搜索。
:::note
搜索功能**仅支持英文搜索**
:::
3. 点击你想要的世界，打开版本列表页面
4. 选择游戏版本并点击打开
:::note
FCL会根据你现在选中的*游戏版本*来挑选合适的*世界的游戏版本*置顶，如图。请优先选择置顶的版本
:::
5. 选择存档将被保存的路径。点击你希望的世界版本，弹出路径选择页面，选择路径。推荐选择希望安装到的Minecraft的**版本文件夹**。
即：`/storage/emulated/0/FCL/.minecraft/versions/[版本名称]`
6. 点击确定路径并开始下载
7. [下一步](#存档的安装)

### 浏览器下载

1. 在MT管理器找到在浏览器下载的存档，通常在`/storage/emulated/0//Download/`
2. 将存档文件移动到`/storage/emulated/0/FCL/.minecraft/versions/[版本名称]`
3. [下一步](#存档的安装)

## 存档的安装
相信你已经把存档文件移动到`/storage/emulated/0/FCL/.minecraft/versions/[版本名称]/`了，那么，现在就是安装的时候了

1. 在MT管理器**左侧窗口**跳转到`/storage/emulated/0/FCL/.minecraft/versions/[版本名称]/`
2. 在MT管理器**右侧窗口**跳转到`/storage/emulated/0/FCL/.minecraft/versions/[版本名称]/saves/`
3. 长按左侧的压缩包，点击解压，打开解压界面
4. 点击**解压到**，勾选**解压到当前目录**、**基于另一窗口**和**解压后删除源文件**，点击确定。
:::note
即使在该路径中不存在`saves`文件夹，解压时MT也会自动帮你创建，所以不用担心跳转到了一个不存在的目录
:::
5. 等待解压完成后，即代表安装存档完成。
