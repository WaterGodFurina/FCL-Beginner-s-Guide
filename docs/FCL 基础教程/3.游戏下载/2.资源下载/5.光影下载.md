# 光影下载
光影的下载安装和模组类似，但是也有一点不同

## 光影加载器

### 光影加载器的选择
首先，我们需要明确需求
- 如果你只是想要开个光源，发一张截图那么推荐**Fabric+Iris(性能更强)**或**原版客户端+OptiFine(更加简单)**
- 如果你想要在玩模组时加光影，那么推荐Forge/NeoForge+Embeddedium
- 如果你注重性能，想要在玩原版时开光影，那么推荐Fabric+Iris

### 光影加载器的下载&安装
- OotiFine:
在下载版本时，点击OptiFine，即可在安装版本时将OptiFine一并安装
- Forge/NeoForge/Fabric+Embeddedium/Iris
因为Embeddedium和Iris都是模组，所以直接参考[模组安装](docs/FCL%20基础教程/3.游戏下载/2.资源下载/2.模组下载.md)

## 光影的下载

### FCL内置下载
1. 打开资源下载页面，点击第一行第六个“**光影**”
2. 选择光影并点击打开到版本选择页面
3. 选择对应的游戏版本
:::note
FCL会根据你的游戏版本自动置顶合适的版本，如没有特殊情况，选择置顶的版本即可
:::
4. 选择光影的版本并点击，一般情况下，最新即可
5. 点击确定开始下载，等待下载结束
6. 点击完成，完成下载

### 浏览器下载
1. 在MT管理器找到在浏览器下载的光影包，通常在`/storage/emulated/0//Download/`
2. 将光影包移动到`/storage/emulated/0/FCL/.minecraft/versions/[版本名称]`
3. 在MT管理器**左侧窗口**跳转到`/storage/emulated/0/FCL/.minecraft/versions/[版本名称]/`
4. 在MT管理器**右侧窗口**跳转到`/storage/emulated/0/FCL/.minecraft/versions/[版本名称]/shaderpacks/`
5. 长按左侧的压缩包，点击**移动**，点击确定
:::note
即使在该路径中不存在`shaderpacks`文件夹，解压时MT也会自动帮你创建，所以不用担心跳转到了一个不存在的目录
:::
