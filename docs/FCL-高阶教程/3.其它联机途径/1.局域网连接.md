# 局域网联机
>MC联机有多种方式，局域网联机是主要的一种。

## 如何进行局域网联机

#### 步骤如下

**1.** 两人（或多人）安装相同版本的MC，安装相同的**客户端需装，服务端需装**的模组，仅**客户端需装**的模组可以不相同

> 但是为了较好的游戏体验，建议所有玩家安装相同的模组。

:::note 客户端需装和服务端需装

这里参考MC百科模组介绍中的运行环境
![模组运行环境](/img/docs/FCL-高阶教程/3.其它联机途径/局域网连接/运行环境.jpg)

:::

**2.** 所有玩家连接相同的网络
:::note
如果无法连接同一网络也是可以局域网联机的，后文有说明。
:::

**3.** 其中一个玩家在单人游戏中进入联机使用的世界，按ESC键，进入游戏菜单，点击`对局域网开放`
![游戏菜单](/img/docs/FCL-高阶教程/3.其它联机途径/局域网连接/游戏菜单.jpg)

**4.** 打开`局域网`开关，在下面可以调整对其他玩家的设置，然后应用更改
![对局域网开放](/img/docs/FCL-高阶教程/3.其它联机途径/局域网连接/对局域网开放.jpg)

**5.** 如果聊天栏出现以下提示，说明局域网联机已经开启
![聊天栏提示](/img/docs/FCL-高阶教程/3.其它联机途径/局域网连接/聊天栏提示.jpg)

**6.** 其他玩家打开多人游戏，一般情况下在列表中会出现联机世界，点击进入世界即可联机。

:::tip 如果多人游戏列表中没有出现联机世界

可以点击下面的直接连接，输入**开局域网的玩家的设备**的`IP地址:端口号`进入游戏，这里的`:`是英文冒号

:::

## 异地组网
>如果无法连接同一网络（朋友不在身边等），也想进行局域网联机，这时候需要一个异地组网软件

以软件`贝锐蒲公英`为例，步骤如下

准备工作： 所有玩家下载`贝锐蒲公英`软件，登录相同的账号并连接。

**1.** 其他玩家在软件的`设备列表`中找到**开局域网的玩家的设备**并点击，查看虚拟IP地址，然后进入多人游戏。

**2.** 点击直接连接，填入`虚拟IP地址:端口号`，进入游戏
