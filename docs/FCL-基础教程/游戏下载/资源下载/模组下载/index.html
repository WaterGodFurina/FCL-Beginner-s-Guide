模组下载与安装

在开始下载前，请确保：你的 FCL 已成功创建并开启了一个游戏版本，且你知道该版本对应的 Minecraft 版本号（例如 1.20.1）。

FCL 支持两种主流的模组安装方式。其中，通过内置市场下载（方法一）最为安全便捷，也是官方推荐的首选方式；如果你有现成的模组文件，则可以使用本地导入（方法二）。

方法一：通过内置模组市场下载（推荐）​

FCL 内置了模组市场，可以直接在应用内浏览、搜索和下载模组，是最便捷的方式。

  1. 打开 FCL，点击左边的第三个类似云朵并带↓箭头图标的按钮

  2. 在顶部切换至 第三个「模组」 选项卡

  3. 点击之后我们就能看到各种各样的模组以及模组左边的有搜索栏，下面还有筛选模组，可以按自己的需求去找模组。

  4. 找到想要的模组后点击进入，选择适配的模组版本，在下载之前要看清楚自己玩的版本有没有装forge以及fabric等，如果是forge或者是fabric就要去下载对应的模组。

备注

FCL会根据你的游戏版本自动置顶合适的版本，如没有特殊情况，选择置顶的版本即可

  1. 点击 「下载」 ，模组将自动安装到对应版本中

提示：启动游戏前，建议在「版本设置」中确认该模组已被勾选启用。

方法二：导入本地模组文件​

如果你已经从mc百科等其他渠道下载了模组 .jar 文件可以通过本地导入功能安装。

  1. 将模组 .jar 文件放入手机任意文件夹。（前提是你能找到，请不要放在Android/data，FCL无法访问，放根目录就行）

  2. 在 FCL 的左边第二个扳手点击，同时再点顶部的第四个「管理模组」 。

  3. 点击左下方添加模组，找到并选择你的 .jar 文件，点击确认。 系统会提示“安装成功”，点击 「确认」 完成安装

提示：下载模组的时候一定一定要看看有没有前置模组。

方法三：手动放置模组文件​

你也可以通过文件管理器手动将模组放入对应目录。

  • 默认全局目录：/storage/emulated/0/FCL/.minecraft/mods/

  • 特定版本目录：/storage/emulated/0/FCL/.minecraft/versions/<版本名>/mods/

注意：全局目录下的模组会被所有未启用版本隔离的版本共享，不同版本间可能出现模组不兼容的问题。建议打开版本隔离，方便管理各个版本模组。

前置条件

在安装模组之前，请确保：

  • 已安装对应的模组加载器（Forge / Fabric / Quilt）

点扳手→ 点自动安装 → 选择「模组加载器」→ 选择 Forge / Fabric / Quilt → 点击安装

  • 模组版本与游戏版本及加载器类型匹配，这一点可以从名字看出来。例如：journeymap-fabric-1.21.1-5.10.1.jar 仅适用于 Fabric + 1.21.1 环境

常见问题

Q：下载的模组在游戏中不生效？

A：检查模组版本是否与当前游戏版本和加载器（Forge/Fabric）匹配；确认在「版本设置」中已勾选该模组。

Q：提示缺少前置模组？

A：部分模组依赖其他模组作为前置，请自行下载，一般情况下mc百科等其他渠道都有进行说明，自觉下载就好。

Q：模组文件应该放在哪里？

A：应放在mods文件夹，一般在版本文件夹里面有，如果没有，大概是游戏没启动或者是没装模组加载器。

Q：没有我要找的模组版本？

A：可以切换下载源试试
