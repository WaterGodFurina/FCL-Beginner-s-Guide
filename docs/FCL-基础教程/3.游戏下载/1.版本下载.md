# 版本下载

FCL内部集成了Minecraft版本的下载功能，只需要在FCL内部就能完成版本下载与安装

## 如何安装

1. 打开下载页面。点击左侧导航栏的**下载“☁️”**按钮，转到下载页面
![1.打开下载页面](/img/docs/FCL%20基础教程/3.游戏下载/1.版本下载/1.打开下载页面.jpg)

:::tip 加载不出来？
版本列表加载不出来可能是由多种原因导致的，笔者目前也不清楚，不过你可以试试清理启动器缓存、更新启动器、使用代理等
:::

2. 选择和筛选版本。在页面**中间第二行**处可以筛选正式版、测试版和远古版，右侧的搜索栏可以**精细搜索**每一个筛选出来的版本
![2.筛选版本](/img/docs/FCL%20基础教程/3.游戏下载/1.版本下载/2.筛选版本.jpg)

:::tip 只能搜索筛选出来的版本
你只能搜索出你已经筛选出来的版本，例如：你**只勾选了正式版**，你将无法搜索出测试版和远古版的版本
:::

3. 配置你想要下载的版本。这里我们以1.20.4为例，点开你想要安装的游戏版本，**中间第二行的空格**处可以修改**版本的名称**，下面可以选择你想要的**模组加载器/光影加载器**，至于该怎么选，会在本文末尾展开
![3.配置版本](/img/docs/FCL%20基础教程/3.游戏下载/1.版本下载/3.配置版本.jpg)

4. 下载并安装版本。**中间第二行右侧**的**下载↓**按钮开始版本的下载和安装，点击**取消**按钮以取消下载
![4.下载版本1](/img/docs/FCL%20基础教程/3.游戏下载/1.版本下载/4.下载版本1.jpg)
![4.下载版本2](/img/docs/FCL%20基础教程/3.游戏下载/1.版本下载/4.下载版本2.jpg)

:::tip 下载/安装的过程中卡住了？
如果是下载过程中卡住了，可以试试切换*下载源*；如果是安装过程中卡住了，可以试试清理缓存+重启启动器
:::
:::danger 不要**删掉FCL的后台**！谨慎**挂后台**！
在下载内容时，如果FCL的后台被删除，下载会被**强行中断**
在下载内容时，FCL被挂在后台，系统分配给FCL的内存可能减少，导致下载变慢
怎么做？可以用系统自带的小窗功能将FCL挂在屏幕上，防止下载中断
:::

5. 等待下载完成，点击“**完成**”，完成版本安装


## 如何选择加载器

我该怎么选择模组加载器？以下的表格可能可以帮助你

| 加载器 | 特点 |
|--------|------|
| Forge | 老牌模组加载器，有大量**旧版本模组**，社区庞大功能丰富但加载较重，**性能一般**，对新版本支持较慢，适合需要大量老牌模组或大型整合包的玩家 |
| Fabric | **轻量级、加载快、优化好**，新版本模组多，旧版支持有限。适合追求**高性能**、玩最新版本的玩家 |
| NeoForge | 较Forge更新速度更快，**性能更优秀**；较Fabric**功能更丰富**。适合玩高版本的模组 |
| Quilt | 是Fabric的一个社区分支，支持绝大多数Fabric模组|
| OptiFine | 性能优化与图形增强模组，能显著提升帧率、支持高清纹理与光影，提供大量可自定义的视觉与性能选项，但在高版本中，兼容性欠佳 |
| LiteLoader | 
| Cleanroom | 是为**1.12.2**打造的现代化Forge分支，通过引入当前最新技术来**提升游戏性能、加载速度及兼容性** |
