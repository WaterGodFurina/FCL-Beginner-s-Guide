# 离线登录

:::tip 专有名词
本文涉及到的专有名词：[ID](/docs/technical-term/ID) 、 [UUID](/docs/technical-term/UUID)
:::

## 什么是离线登录？

离线登录是一种能让用户在**离线状态**（不连接[互联网](https://baike.baidu.com/item/%E4%BA%92%E8%81%94%E7%BD%91/199186)）的情况下登录并游玩Minecraft的一种登录方式

## 技术原理：

不向**Mojang**的验证服务器发送请求，而是直接由启动器用*自定义*的**用户名ID和UUID**启动游戏，跳过**身份核验**

## 在FCL内使用离线登录

1. 打开账户页面。在FCL主页点击右侧“**头像**”
![1.打开添加账户页面](/img/docs/FCL%20基础教程/2.账号登录/1.离线登录/1.打开添加账户页面.jpg)

2. 添加离线帐户。在**创建账户**中找到并点击“**离线账户**”
![2.点击添加离线帐户](/img/docs/FCL%20基础教程/2.账号登录/1.离线登录/2.点击添加离线帐户.jpg)

3. 输入一个自定义的用户名
![3.输入用户名](/img/docs/FCL%20基础教程/2.账号登录/1.离线登录/3.输入用户名.jpg)

:::tip 用户名的要求
你可以使用 `英文`、`数字`、`下划线_` 来创建你的用户名，并且`长度必须在3~16个字符之间`。
:::
:::danger 不要使用中文名！
在创建账户时，不要使用**中文字符**、**感叹号!**和**空格**和特殊字符等，否则可能导致一些游玩时的*问题*甚至是**崩溃**！
:::

4.点击**登录**即可完成创建离线账户。
![4.确认添加账户](/img/docs/FCL%20基础教程/2.账号登录/1.离线登录/4.确认添加账户.jpg)
![5.添加完成](/img/docs/FCL%20基础教程/2.账号登录/1.离线登录/5.添加完成.jpg)

:::tip 支持正版账户
离线账户虽好，但是会受到一些限制：

- 进不了正版服务器（如:[Hypixel](https://hypixel.net/)、[Shotbow](https://shotbow.net/)、[Mineplex](https://mineplex.com/home/)）
- 更换后的皮肤只在单人游戏下生效，多人游戏下无效
- 无法和他人添加为好友（26.2版本+）

如果你的经济实力允许，可以支持一下正版。
:::
