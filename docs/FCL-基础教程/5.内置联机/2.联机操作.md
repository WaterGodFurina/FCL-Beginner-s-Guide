#联机操作（内置联机）

>FCL内置联机（陶瓦联机）*不需要*同一网络。双方**网络质量**与**最终联机体验**有较大关系
在开始前，请确保：
- 联机双方的FCL启动器版本在1.2.7.1及以上
- 联机双方所加的模组、游戏版本、模组加载器、或整合包**全部一致**

## 开启多人联机功能

1. 打开FCL，“**点击**”左侧第五个多人联机（类似三个方块连接的按钮）

2. 在顶部有“**启用多人联机**”这个功能，打开

- 如果你是什么都不懂的小白请点击 [基础教程](#基础教程)
- 如果你略懂一些基础请点击 [进阶教程](#进阶教程)
- 如果你基本都懂了请点击 [高阶教程](#满级大佬教程)或[高阶联机](/docs/FCL-高阶教程/其它联机途径/内网穿透连接)

## 基础教程

### 房间的创建

1. 打开FCL，点击右侧“**启动**”，启动游戏，等待进入(启动时至少要有一个游戏版本：[版本下载](/docs/FCL-基础教程/游戏下载/版本下载)

2. 点击“**单人游戏**"进入你想要进行联机的存档（如果没有那就创建一个）

3. 按**ESC**打开游戏暂停菜单，点击“**对局域网开放**”，点击下方偏左的“**创建局域网世界**”

4. 点击FCL菜单（圆圈中的附魔台图标），在右侧设置找到联机菜单并打开，点击“**我想当房主**”。开始扫描游戏端口

:::note
若弹出提示要求**申请VPN权限**，授予即可，否则无法正常使用
:::

5. 等待片刻将自动复制邀请码，若没有自动复制也可以点击“**复制邀请码**”手动复制

6. 将复制的邀请码通过*各种方式*发送给要联机的好友后回到游戏即可

:::note
若要关闭房间，打开联机菜单点击“**退出**”即可
:::

### 房间的连接

1. 打开FCL，点击右侧“**启动**”，启动游戏，等待进入（需要和房主游戏版本一致）

2. 点击FCL菜单（圆圈中的附魔台图标），在右侧设置找到联机菜单并打开，点击“**我想当房客**”

3. 将房主给的邀请码复制后粘贴进去，点击“**确定**”，等待加入房间，点击确定回到游戏

:::note
若弹出提示要求**申请VPN权限**，授予即可，否则无法正常使用
:::

4. 点击“**多人游戏**”，刷新，会出现新的局域网世界，双击等待进入即可

:::note
若要退出房间可打开联机菜单点击“**退出**”即可
:::

>不常见问题

:::note Q：我找不到**ESC**怎么办？
- 点击FCL菜单（圆圈中的附魔台图标），（新版请按音量键），查看左侧控制**隐藏所有控件**是否开启，请关闭试试
- 点击FCL菜单（圆圈中的附魔台图标），查看左侧控制打开**编辑模式**，下滑找到**添加按键**并点击，配置ESC按键即可：[按键布局](/docs/FCL-高阶教程/按键制作/)
:::

:::note Q：我找不到FCL菜单怎么办？
A：按手机的音量键，即可打开FCL菜单，查看左侧控制**隐藏所有控件**是否开启，请关闭试试
:::

:::note Q：房间进不去怎么办
A：检查房间是否关闭、双方的网络是否正常
:::

## 进阶教程

### 房间的创建

1. 启动游戏，进入联机存档，开启局域网联机

2. 打开FCL菜单，打开联机菜单，选择我要当房主

:::note
若弹出提示要求**申请VPN权限**，授予即可，否则无法正常使用
:::

3. 将复制的邀请码发给联机的好友

### 房间的连接

1. 启动游戏，打开FCL菜单，打开联机菜单，选择我要当房客，输入邀请码

2. 点击多人联机，刷新出现局域网世界，双击进入即可

:::note
若弹出提示要求**申请VPN权限**，授予即可，否则无法正常使用
:::
