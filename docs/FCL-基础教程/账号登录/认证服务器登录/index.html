FCL 启动器外置账号从注册到登录完整指南

注意：本文所说的外置登录或第三方登录，均指认证服务器登录

很多玩家在游玩第三方服务器或使用离线皮肤时，需要用到外置登录（Yggdrasil 认证）。下面带你从 0 开始，走完注册第三方账号到成功在FCL启动器登录账号的每一步。

第一步：注册第三方皮肤站账号​

目前国内主流的第三方外置登录皮肤站有 LittleSkin、Blessing Skin 等。这里以 LittleSkin 为例：

  1. 打开你的浏览器，访问 LittleSkin 官方网站

  2. 点击网站正中央的 “现在注册” 按钮。

  3. 填写注册信息：

  • 邮箱：必须使用真实有效的邮箱（用于接收验证邮件）。
  • 密码：按要求设置复杂度足够的密码。
  • 重复密码：重复输入密码即可。
  • 昵称：这是你在 LittleSkin 的名字，并不是游戏里的名字。
  • 验证码：看图输入即可

  1. 提交注册后，前往你的邮箱查收验证邮件，并点击邮件里的激活链接。

  2. 回到网站，使用你的邮箱和密码登录皮肤站后台，确认账号已激活。至此注册账号已成功

第二步：创建角色​

  1. 进入后台之后，需要点左上角的三条杠，找到并点击角色管理

  2. 点击之后我们看到的是PID、角色、操作、以及添加新角色，下方便是正版验证&绑定，拥有正版账号的玩家可以去绑定，按提示走即可，这里不多多介绍。

  3. 点击添加新角色，会让你输入角色名，角色名不能带中文，可以使用大小写英文字母、数字和部分特殊符号，角色名最少要求 3 个字符，最多不超过 16 个字符。(中文输入上去了联机将无法进行，请按以上要求取名)

  4. 添加完成之后，如果你不装皮肤和披风的话，你可以去看“第四步 找到并输入认证服务器地址”了。

第三步：如何装皮肤以及披风​

  1. 在添加完角色之后，我们可以随心所欲的为自己装皮肤以及披风了！

  2. 依旧是点击左上角的三条杠，下滑找到并点击皮肤库

  3. 点击之后，我们可以挑选自己的皮肤以及披风了，在找到喜欢的皮肤以及披风之后，点击它，点击之后我们将看到皮肤或披风预览、添加衣柜、设为头像、下载、举报等，下面是关于皮肤或者是披风的详细信息。（搜索栏旁边可切换披风或者是皮肤）

  4. 点击添加衣柜之后，会有一个弹窗，是让你起名，这里可起可不起，直接确定即可，ok！会有一个弹窗提示你已添加至衣柜，这里不用退，因为我们的添加衣柜变为立即使用了，请点击它。

  5. 点击之后，有一个弹窗说给哪个角色使用？请点击我们所刚刚创建的角色，点击完成之后我们的皮肤就已经成为装在了这个角色。（披风与4、5步骤是一样的）

第四步：找到并复制认证服务器地址​

接下来你需要找到外置登录的 API 地址：

  1. 你需要找到外置登录的 API 地址

  2. 登录皮肤站后，在仪表盘里下滑找到并点击“Yggdrasil 外置登录”入口。

  3. 点击之后下滑找到Yggdrasil API 地址，在下方点击并复制API地址

  4. 至此你已成功找到并复制了API地址。

第五步：在 FCL 启动器进行登录​

现在账号有了，API 地址也有了，接下来在 FCL 启动器里操作：

  1. 打开 FCL 启动器，进入主界面，点击 “右边的小人头像”，进入账号管理。

  2. 点击左下角的 “认证服务器”。

  3. 点击之后会弹出窗将第四步复制好的 API 地址 粘贴到弹出的输入框中，点击下一步，又弹出了窗，请点击确认。

  4. 此时启动器会在左边的微软下面添加littleskin，点击它，请务必使用你在皮肤站注册时的邮箱（或用户名）以及密码进行登录。点击“登录”。

  5. 登录之后，你的账号就会保存到 FCL 的账号列表中。

⚠️ 常见问题（注册与登录）​

Q：为什么我在皮肤站注册账号时，收不到验证邮件？

A：请检查是否误入了垃圾邮件箱。如果还是收不到，可能是该皮肤站邮箱服务异常，建议换个时间段重试，或联系皮肤站管理员。

Q：在 FCL 登录时，提示“账户不存在”或“密码错误”？

A：请注意，FCL 登录窗口输入的账号密码，必须是皮肤站官网的账号密码也就是邮箱+密码，如果确认官网能登录，检查是否有空格或大小写错误。
