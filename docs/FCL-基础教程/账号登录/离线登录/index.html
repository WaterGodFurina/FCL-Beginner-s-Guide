离线登录

专有名词

本文涉及到的专有名词：用户名 ID 、 唯一标识符 UUID

什么是离线登录？​

离线登录是一种能让用户在离线状态（不连接互联网）的情况下登录并游玩Minecraft的一种登录方式

技术原理：​

不向Mojang的验证服务器发送请求，而是直接由启动器用自定义的用户名ID和UUID启动游戏，跳过身份核验

在FCL内使用离线登录​

  1. 打开账户页面。在FCL主页点击右侧“头像”

  2. 添加离线帐户。在创建账户中找到并点击“离线账户”

  3. 输入一个自定义的用户名

用户名的要求

你可以使用 英文、数字、下划线_ 来创建你的用户名，并且长度必须在3~16个字符之间。

不要使用中文名！

在创建账户时，不要使用中文字符、感叹号!和空格和特殊字符等，否则可能导致一些游玩时的问题甚至是崩溃！

4.点击登录即可完成创建离线账户。

支持正版账户

离线账户虽好，但是会受到一些限制：

  • 进不了正版服务器（如:Hypixel、Shotbow、Mineplex）
  • 更换后的皮肤只在单人游戏下生效，多人游戏下无效
  • 无法和他人添加为好友（26.2版本+）

如果你的经济实力允许，可以支持一下正版。
