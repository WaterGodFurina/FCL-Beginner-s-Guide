# Minecraft 玩家名片：用户名 ID

欢迎来到 Minecraft 世界！当你第一次进入游戏时，系统会让你输入一个名字，这个名字就是你的**用户名 ID（Username）**。

对于刚接触 MC 的新手玩家来说，用户名是你最直观的身份标志。今天我们就来了解一下这个天天见的老朋友。

## 什么是用户名 ID？

简单来说，用户名就是你在游戏里头顶上飘着的那个名字，也是其他玩家在游戏里称呼你的方式。

无论是在单机游戏里按 `Tab` 键查看玩家列表，还是在多人服务器里聊天，大家看到的都是你的用户名。

:::tip 打个比方
用户名就像是你的**衣服**或者**网名**。今天你可以穿红衣服叫“Steve”，明天你可以换身衣服改名叫“Alex”。大家平时怎么称呼你，看的就是这件衣服。
:::

## 用户名的特点

- **直观可见**：任何人都能在游戏里直接看到你的名字。
- **可更改性**：如果你玩的是正版 Java 版，每隔一段时间你是可以修改用户名的；基岩版也可以修改 Gamertag。
- **不绝对唯一**：在整个 MC 历史中，可能有不同的玩家先后使用过同一个名字（比如很多人叫 "Herobrine"）。即使在同一个服务器里，如果服务器允许，也可能出现名字相似的情况。

## 离线服务器中的特殊规则

如果你玩的是“离线模式”（也就是通常说的盗版服）服务器，你的用户名直接决定了你能否登录。

在这类服务器中，你只要在启动器里随便填一个名字（比如 "Notch"），就能以这个名字进服。但也正因为如此，离线服只能靠名字来简单区分玩家，如果你改了名字，服务器就会把你当成新人。

:::caution 新手注意
在离线服务器中，**千万不要太随意地更改用户名**！因为服务器是靠名字来保存你的背包物品和建筑的，名字一改，你的东西可能就找不到了。
:::

## 总结

用户名 ID 是你在 Minecraft 社交的“外衣”，它代表了你的形象。选一个你喜欢的名字，开始你的冒险吧！
